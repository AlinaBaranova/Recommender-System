Topics of the NYPL Digital Collections

About

This is a recommender system for topics of the New York Public Library Digital Collections. The aim of the project is to provide a tool for suggesting the topics that may interest user, relying only on their present search query. The algorithm uses several techniques to expand user's query and recommend them topics they would not find using key word search on the website of the NYPL Digital Collections.
